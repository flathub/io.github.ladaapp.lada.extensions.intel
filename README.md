This project is used to build and publish the Flatpak Extension for Intel GPU support of the app [Lada](https://codeberg.org/ladaapp/lada) to [Flathub](https://flathub.org/en/apps/io.github.ladaapp.lada).

You can find more details about the packaging process in the upstream repository in the [packaging](https://codeberg.org/ladaapp/lada/src/branch/main/packaging) folder.
